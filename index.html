Backup & Restore
Backup (Export)
Click Backup in the sidebar to download all your data as a .json file. Keep this file safe — it contains all projects, requests, timelines, and settings.
Restore (Import)
Click Restore in the sidebar and choose a .json backup file. Warning: this will replace all current data after confirmation.

App Purpose
Priora helps you track the progress of Projects and Requests with structured follow-ups. It gives you full visibility into what is active, overdue, or waiting — without any backend or internet connection required.
Full Guide
Status System
Every item has one of three statuses:
Action Required — Amber. Something requires your immediate action.
Waiting Feedback — Blue. Waiting for response from another party.
On Track — Blue. Progressing normally, no action needed.
Aging Dot (Time-based color)
Each item has a small dot before its name. The color reflects time since the last update (not creation date):
Urgent (override) — always red
On Track (override) — always blue
0–2 days — muted blue (new)
3–7 days — soft yellow
8–14 days — soft orange
14+ days — stronger red (old item needs attention) All colors are muted — no sharp or glowing tones. No mixing. One dot per item only.
Urgent Items
Mark any item as Urgent using the triangle button in the action column. Urgent items: show a blinking red triangle before the name, appear in the Urgent stat card and the Overdue & Urgent section, always show a red aging dot (overrides age logic), require daily follow-up attention.
On Track Behavior
Items marked "On Track" always show a blue aging dot, regardless of how old they are. This means On Track is a positive override — it suppresses the aging warning and signals the item is healthy.
Timeline & Updates
Each item has an update log (timeline). Click the item name to open it. Each entry shows the date and text in one line. You can edit or delete any entry using the icons on the right. Dates are shown only — no time is displayed. When editing an entry, you can change the date, time, and text.
Reports
Generate PDF reports from inside any item (click the report icon in the header). You can also generate global reports: All items, Active only, or Completed only. Reports open in a new tab and trigger the print dialog automatically.
Date Handling
All dates are shown in DD/MM/YYYY format. Dates are never colored — only the aging dot before the item name reflects time status. This keeps the table clean and the dot as the single source of visual urgency.
